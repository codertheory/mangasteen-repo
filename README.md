# mangasteen-repo

A source extension repository for **Mangasteen**, a manga reader built with Kotlin Multiplatform (Android/iOS).

## Overview

Extensions implement the parsing logic for a specific manga source. The Mangasteen app embeds a JavaScript engine that loads and calls these extensions at runtime to search for manga, fetch details, and retrieve chapter/page data.

## Creating Your Own Extension

Want to build an extension for a different manga source? Check out the official extension template:

https://github.com/codertheory/Mangasteen-extension-template
